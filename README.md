A simple CLI for fast development of front end

## useage

``` init <type> <name> ```

**```yt-cli init page test```**
current support page type

will create

```
- test
    - Test.js
    - testReducer.js
```

**```yt-cli init comp test```**

option
- js: JavaScript (default)
- ts: typescript

create:

```
- test
    - index.tsx
    - index.styl
```
